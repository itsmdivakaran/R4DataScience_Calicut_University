The book is authored using **Quarto** (`.qmd` files) and is organized as follows:

| Section | Chapter / File | Description |
| :--- | :--- | :--- |
| **Front Matter** | `index.qmd`, `Author.qmd` | Introduction to the course and authorship. |
| **Foundations** | `03-introduction` to `06-datastructures` | Setting up R, environment, and understanding core data types. |
| **Data Wrangling** | `07-data-import` to `10-tidyr` | The Tidyverse approach to cleaning and reshaping data. |
| **Visualization** | `11a-basicplots` & `11b-ggplot2` | Mastering the Grammar of Graphics. |
| **Advanced Tools** | `12-stringr` to `15.5-forcats` | Strings, Dates, Functional Programming, and Factors. |
| **Statistics** | `16-statistics` to `19-regression` | Descriptive stats, Probability, Inference, and Modeling. |
